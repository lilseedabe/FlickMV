# 🚀 FlickMV Enhanced - クイックスタートガイド\n\n## ⚡ 3分で始める音響同期ビデオ編集\n\n### 📋 事前準備\n\n✅ **必要なもの**\n- 音楽ファイル（MP3, WAV形式）\n- 画像・動画素材\n- モダンブラウザ（Chrome, Firefox, Safari推奨）\n\n### 🎯 Step 1: セットアップ（1分）\n\n```bash\n# プロジェクトを起動\nnpm run dev\n\n# 拡張エディターにアクセス\n# http://localhost:3000/editor-v2\n```\n\n### 🎵 Step 2: BPM検出（1分）\n\n1. **音楽アップロード**\n   ```\n   左パネル → \"メディア\" → \"ファイルをアップロード\"\n   ```\n\n2. **BPM検出実行**\n   ```\n   \"BPMを検出\" ボタンをクリック\n   → 10-30秒で完了\n   → ヘッダーにBPM表示\n   ```\n\n3. **結果確認**\n   - ✅ BPM: 楽曲のテンポ\n   - ✅ 信頼度: 70%以上が理想\n   - ✅ ビートマーカー: タイムライン上の紫線\n\n### 🎬 Step 3: 素材配置とスナップ（1分）\n\n1. **画像・動画アップロード**\n   ```\n   メディアパネル → 素材をドラッグ&ドロップ\n   ```\n\n2. **ビートスナップ有効化**\n   ```\n   タイムライン → \"スナップ\" ボタンをON\n   ```\n\n3. **クリップ配置**\n   ```\n   素材をタイムラインにドラッグ\n   → 自動的にビートにスナップ\n   → 黄色い予告線で確認\n   ```\n\n### ⚡ ワンクリック機能\n\n#### 🎨 プリセット適用（10秒）\n```\n左パネル → \"プリセット\" → \"Summer Vibes\" → \"適用\"\n```\n\n#### 🪄 自動編集レシピ（10秒）\n```\nクリップ選択 → \"レシピ\" → \"ビートカット\" → \"適用\"\n```\n\n#### 📱 SNS最適化（10秒）\n```\n\"SNS\" → \"Instagram リール\" → \"最適化実行\"\n```\n\n### 🎉 完成！\n\n```\n\"エクスポート\" ボタン → 高品質なミュージックビデオが完成\n```\n\n---\n\n## 💡 最初に試すべき機能\n\n### 1. 🎵 EDM楽曲の場合\n```\n1.
