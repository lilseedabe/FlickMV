# FlickMV 音声解析機能 - セットアップ＆使用方法ガイド## 🎯 新機能概要**2025年8月最新版**では、**MoonshotAI Kimi (kimi-k2-instruct)** を使用した高精度な音声解析・MVプロンプト生成機能を搭載し、**無料プランでも利用可能**になりました。### ✨ 機能詳細1. **音声テキスト化**: Groq Whisper-large-v3による高精度な日本語音声認識2. **AI分析**: **MoonshotAI Kimi**による創造的な歌詞・感情分析3. **シーン分割**: 音声のタイムスタンプに基づいたシーン自動分割4. **プロンプト生成**: 各シーンに適した映像生成プロンプトの提案5. **手動編集**: 生成されたプロンプトの編集・カスタマイズ6. **🆓 全プラン対応**: 無料プランでも月2回まで利用可能## 💎 AI技術詳細### 音声認識エンジン- **Groq Whisper-large-v3**: 業界最高レベルの日本語音声認識- **精度**: 99%以上の高精度テキスト化- **対応言語**: 日本語特化（将来的に多言語対応予定）### プロンプト生成AI- **MoonshotAI Kimi (kimi-k2-instruct)**: 創造性に特化したAIモデル- **フォールバック**: Groq LLaMA 3.1 70B（Kimiが利用不可の場合）- **出力品質**: 映像制作プロレベルの詳細なプロンプト生成## 🛠 セットアップ### 1. Groq APIキーの取得1. [Groq Console](https://console.groq.com)にアクセス2. アカウント作成・ログイン3. API Keysセクションから新しいAPIキーを作成4. APIキーをコピー（後で使用）### 2. 環境変数の設定#### サーバー側設定 (.env)```bash# AI ServicesGROQ_API_KEY=your-groq-api-key-here```### 3. 依存関係のインストール#### サーバー側```bashcd servernpm install axios form-data```### 4. サーバー再起動```bashcd servernpm run dev```## 📊 プラン別利用制限### 🆓 フリープラン（無料）- 音声解析: 月2回- プロンプト再生成: 月5回- 動画エクスポート: 月3回- 解像度: 720p- 透かし: あり### 💼 ベーシックプラン（月額 ¥980）- 音声解析: 月8回- プロンプト再生成: 月20回- 動画エクスポート: 月15回- 解像度: 1080p- 透かし: あり### 🚀 プロプラン（月額 ¥1,980）- 音声解析: 月25回- プロンプト再生成: 月75回- 動画エクスポート: 月50回- 解像度: 4K- 透かし: 削除可能### 👑 プレミアムプラン（月額 ¥2,980）- 音声解析: 無制限- プロンプト再生成: 無制限- 動画エクスポート: 無制限- 解像度: 4K- 透かし: 削除可能## 🎵 使用方法### 1. 音声ファイルのアップロード1. エディターページを開く2. 左パネルの「メディア」タブを選択3. 音声ファイル（.mp3, .wav, .m4a等）をアップロード### 2. 音声解析の実行1. アップロードされた音声ファイルにマウスオーバー2. 表示される紫色の**脳アイコン**をクリック3. 解析オプションを設定：   - **音楽ジャンル**: ポップス、ロック、バラード等   - **雰囲気**: 明るい、感動的、力強い等   - **映像スタイル**: モダン、シネマティック、アニメ風等4. 「音声解析開始」ボタンをクリック### 3. 解析結果の確認解析完了後、以下の情報が表示されます：#### 📝 音声テキスト化結果- 音声から変換された歌詞テキスト- セグメント別のタイムスタンプ付きテキスト#### 🎨 全体テーマ- 楽曲全体のコンセプトとテーマ提案#### 🎬 シーン別MVプロンプト各シーンごとに以下の情報を提供：- **タイムスタンプ**: 開始・終了時間- **歌詞**: 該当部分の歌詞- **映像プロンプト**: 詳細な映像生成指示- **雰囲気**: シーンの感情的トーン- **カラーパレット**: 推奨色彩- **キーワード**: 関連キーワード#### 💡 制作提案- カメラワーク提案- エフェクト提案- トランジション提案- 全体構成提案### 4. プロンプトの編集・カスタマイズ1. 各シーンの**編集アイコン**をクリック2. 映像プロンプトを自由に編集3. 「変更を保存」ボタンで保存### 5. プロンプトの再生成1. 解析オプションを変更2. 「プロンプト再生成」ボタンをクリック3. 新しい設定でプロンプトを再生成## 🎮 実用的な使用例### ケース1: ポップスソング```設定:- ジャンル: ポップス- 雰囲気: 明るい- スタイル: モダン生成例（MoonshotAI Kimiによる）:- イントロ: \"カラフルな背景に踊る光の粒子、明るく元気な雰囲気、若々しいエネルギーが溢れる映像\"- サビ: \"歌手が笑顔で歌う様子、虹色のライティング効果、観客との一体感を演出\"```### ケース2: バラード```設定:- ジャンル: バラード- 雰囲気: 感動的- スタイル: シネマティック生成例（MoonshotAI Kimiによる）:- 冒頭: \"夕焼けを背景にした静かな街並み、温かみのある照明、ノスタルジックな色調\"- 転調部: \"感情が高まる瞬間、カメラが緩やかにズームイン、表情のクローズアップ\"```## ⚙️ 詳細設定オプション### 音楽ジャンル選択- **自動判定**: AIが音楽を分析して自動判定- **ポップス**: 明るく親しみやすい映像- **ロック**: エネルギッシュで力強い映像- **バラード**: 感情的で落ち着いた映像- **ヒップホップ**: リズミカルでストリート感のある映像- **エレクトロニック**: 未来的でデジタル感のある映像- **ジャズ**: 洗練されたクラシカルな映像- **クラシック**: 上品で格調高い映像### 雰囲気設定- **自動判定**: 歌詞の内容から自動判定- **明るい**: ポジティブで元気な映像- **感動的**: 心に響く感情的な映像- **力強い**: ダイナミックで迫力のある映像- **穏やか**: 静かで癒やし系の映像- **ドラマチック**: 劇的で印象的な映像- **ノスタルジック**: 懐かしさを感じる映像### 映像スタイル- **モダン**: 現代的でスタイリッシュ- **シネマティック**: 映画的でドラマチック- **アニメ風**: アニメーション的表現- **リアリスティック**: 現実的で自然な表現- **アート**: 芸術的で抽象的- **レトロ**: 懐かしい雰囲気## 🔧 トラブルシューティング### よくある問題と解決方法#### 1. 「Audio analysis service is not configured」エラー**原因**: Groq APIキーが設定されていない**解決方法**: ```bash# .envファイルに追加GROQ_API_KEY=your-actual-api-key```#### 2. 解析が開始されない**原因**: 利用制限に達している**解決方法**: - 月間制限をチェック（無料プランは月2回）- プランアップグレードを検討- 対応形式: MP3, WAV, M4A- ファイルサイズ: 50MB以下#### 3. 解析がタイムアウトする**原因**: 音声ファイルが長すぎる**解決方法**: - 5分以下の音声ファイルを使用- より短い音声に分割#### 4. 日本語認識精度が低い**原因**: 音質やノイズの問題**解決方法**: - 高音質な音声ファイルを使用- ノイズを除去- 明瞭な発音の音声を使用#### 5. MoonshotAI Kimiモデルエラー**原因**: モデルが一時的に利用不可**解決方法**: - 自動的にLLaMA 3.1 70Bにフォールバック- 少し時間をおいて再試行- サーバーログで詳細確認## 🔍 利用状況の確認### 使用回数の確認方法1. エディター画面右上の使用回数表示2. `/api/media/usage-stats` エンドポイントで詳細確認3. プランアップグレード時の残り回数表示### リセット日程- **月間制限**: 毎月1日にリセット- **リアルタイム表示**: 残り回数が常に更新## 💡 活用のコツ### 1. 効果的なプロンプト生成のために- **音声品質**: クリアで聞き取りやすい音声を使用- **ジャンル設定**: 正確なジャンル選択で適切なプロンプト生成- **雰囲気調整**: 楽曲の感情に合った雰囲気設定### 2. プロンプト編集のベストプラクティス- **具体性**: 抽象的ではなく具体的な描写を追加- **視覚要素**: 色彩、照明、構図を明確に指定- **感情表現**: 感情や雰囲気を織り込む### 3. 効率的なワークフロー1. 複数の設定で解析を試行2. 気に入ったプロンプトをベースに編集3. シーン間の連続性を考慮4. 全体的な統一感を保持## 🚀 今後の予定### 近日追加予定 (1-2ヶ月)1. **多言語対応**: 英語、中国語、韓国語の音声解析2. **感情分析強化**: より詳細な感情分析とムード検出3. **音楽分析**: BPM、キー、コード進行の自動検出4. **テンプレート機能**: ジャンル別プロンプトテンプレート5. **バッチ処理**: 複数ファイルの一括解析### 中期計画 (3-6ヶ月)1. **リアルタイム解析**: ライブ音声のリアルタイム処理2. **AI映像生成連携**: プロンプトから直接映像生成3. **協調編集**: チームでのプロンプト共同編集4. **A/Bテスト**: 複数プロンプトバリエーションの比較5. **ボイス指示**: 音声でのプロンプト編集指示### 長期ビジョン (6ヶ月以上)1. **独自AIモデル**: FlickMV専用の映像プロンプト生成モデル2. **GPU処理対応**: 高速大容量ファイル処理3. **プラグインシステム**: サードパーティ拡張機能4. **API公開**: 外部サービスとの連携5. **VR/AR対応**: 3D・没入型MV制作## 📞 サポート### お問い合わせ- **技術的問題**: support@flickmv.com- **機能リクエスト**: features@flickmv.com- **ドキュメント**: https://docs.flickmv.com### コミュニティ- **Discord**: https://discord.gg/flickmv- **GitHub**: https://github.com/flickmv/flickmv- **YouTube**: チュートリアル動画を公開中---**最終更新**: 2025年8月18日  **バージョン**: 1.2.0 - MoonshotAI Kimi Edition  **作成者**: FlickMV開発チーム
